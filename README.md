# saolei
扫雷---Mac and Windows

# Installation

## Mac
```python
pip install pygame
pip install pyinstaller
pyinstaller minesweeper_mac.spec
dist/saolei
```

## Windows
```python
pip install pygame
pip install pyinstaller
pyinstaller minesweeper.spec
dist\saolei
```

在dist文件夹下查找saolei.app或saolei.exe即可, 资源文件已经全部打包，可以随意移动位置
